## Product Hunt Comments

### First Comment (Maker's Story)

Hi Product Hunt! I'm the maker behind OmniPost.

I started building this after watching a friend -- a freelance content creator managing four brand accounts -- spend her entire Sunday afternoon reformatting the same five posts for different platforms. Same message, four different formats, four different image sizes, four different character limits. Every week.

That felt like a problem software should have solved by now. Tools like Buffer and Hootsuite handle scheduling well, but they don't adapt your content. You still do the reformatting manually.

OmniPost takes a different approach. Write your content once in a single editor, and the AI engine rewrites it for each connected platform. It adjusts tone (professional for LinkedIn, conversational for Twitter), respects character limits, and even generates platform-optimized images. Then you review, schedule, and publish -- all from one dashboard.

The tech stack is Next.js 16, React 19, TypeScript, and PostgreSQL. The entire project is MIT-licensed and open source.

This is an alpha release. There are rough edges. But the core flow works: write once, adapt automatically, publish everywhere.

I'd love to hear what you think. What platforms should we add next? What features would make this indispensable for your workflow?
